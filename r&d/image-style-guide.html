6. Media Side-by-Side (Standard)

Image Left, Text Right

.media-side.within-text: This uses a CSS Grid layout. The image is on the left (approx 40%) and text on the right. It is not a float; it is a grid column.

7. Media Side-by-Side (Reverse)

Text Left, Image Right

.media-side.within-text.reverse: The .reverse class swaps the order. Text takes the larger column on the left, image takes the smaller column on the right.

8. Media Side-by-Side (Seam)

Tight Gutter

.media-side.within-text.seam: The .seam class removes the large gap between image and text, creating a tighter magazine-like feel.

9. Media Side-by-Side (Equal Height)

Equal Height

.media-side.within-text.equal-height: The image will stretch (cover) to match the height of this text block perfectly. Useful if you have a long caption or sidebar text.
